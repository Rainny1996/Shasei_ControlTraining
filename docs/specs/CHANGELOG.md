# 需求文档 Changelog

> 本文件记录「男性控制训练」iOS 应用**需求文档**的版本变更（与 `requirements.md` 同步）。
> 应用代码版本另见 `preview/versions/changelog.md`（UI 预览）、发布归档见 `archive/`。

## [v2.1] - 2026-07-12（训练计划增强版）

> 基于 v2.0（重写补全版）扩展，新增训练计划模块三项功能需求与可量化 AC，并更新需求间数据流与验收追溯矩阵。

### Added（新增需求）
- **需求 10 - 自定义训练计划**
  - 支持「选模板再改」与「空白自建」两种组建方式，共用同一编辑器
  - 可调维度：训练方法（增删/替换）、每周训练天数与具体训练日期
  - 可将自定义结果保存为「我的模板」（本地 Core Data 持久化，支持复用/删除）
  - 对应 **AC-10.1 ~ AC-10.7**
- **需求 11 - 当前计划逐条编辑**
  - 进入编辑模式逐条修改：替换方法、调整单次时长、调整所在日期
  - 支持增删某天训练项目；编辑草稿可取消（不落库），保存时校验
  - 复用既有 `PlanRepository` / `PlanItem`，保留「智能调整」入口
  - 对应 **AC-11.1 ~ AC-11.7**
- **需求 12 - 今日训练动作直达陪练**
  - 点击今日训练动作行（非完成按钮）进入动作详情页
  - 详情页含方法说明与「开始陪练」按钮，进入既有 `CoachView` 并携带 `planItemId`
  - 陪练正常完成后自动 `markPlanItemCompleted` 并刷新；自行退出不勾选
  - 对应 **AC-12.1 ~ AC-12.6**

### Changed（文档变更）
- `requirements.md` 版本号 `v2.0` → `v2.1`，日期 `2026-07-11` → `2026-07-12`
- §9 需求间数据流转图补充需求 10/11/12 的流向
- §10 验收追溯矩阵补充三个新需求 → 实现模块（`PlanBuilderView` / `UserPlanTemplate` / `PlanEditView` / `PlanItemDetailView` / `CoachView(planItemId)`）映射
- 文档尾部补充 v2.1 新增说明

### 范围边界（本次未涉及）
- 自定义组建阶段不暴露单次时长/强度/周期调整（时长留待需求 11，强度/周期沿用既有映射）
- 未改动 `adjustPlanIfNeeded`（智能调整）逻辑
- 未触及隐私（需求 7）、合规（§6）、数据管理（需求 8）等横切模块
