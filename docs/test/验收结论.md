## 二、实现审查结论（关键改动）

验收前发现并修复了两处**实现缺口**（否则全模块无法编译、测试无法运行）：

1. **`PlanService` 不可注入**：原 `private init()` 导致「我的模板」持久化只能走生产库，AC-10.5 无法在内存库单测。已改为 `init(dataController: DataController = .shared)`，`saveUserTemplate/loadUserTemplates` 改经 `PlanRepository(dataController:)`；`shared` 默认生产库，**行为不变**，需求 3 回归安全。
2. **`PlanViewModel.customPlanDraft` 缺失**：`PlanBuilderView` 大量引用该属性但实现未声明，导致 `ControlTraining` 模块**无法编译**。已补 `@Published var customPlanDraft = PlanDraft()`。

上述改动均为**最小、非破坏**改造，未触碰 `regeneratePlan/adjustPlanIfNeeded/createPlanFromTemplate` 等既有逻辑。
