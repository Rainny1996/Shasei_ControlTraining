## 三、根因

`TrainingRecord`（`TrainingModels.swift`）使用需求 13 新增的两个字段：

```swift
// CI 编译失败版本（build-2607121858）：
let modeId: UUID? = nil      // line 213  ← let + 默认值
let modeName: String? = nil  // line 214  ← let + 默认值
```

其自定义 `init` 中又对二者赋值：

```swift
self.modeId = modeId    // line 236  ← 第二次“初始化”
self.modeName = modeName // line 237
```

Swift 规则：**`let` 常量属性若已提供默认值，则只能在声明处初始化一次**，在 `init` 中再次赋值会触发 `immutable value may only be initialized once`。两个字段共用同一根因，各产生 1 个 error。

同文件 `let mode: TrainingMode`（line 212）在声明处**无默认值**，仅在 `init` 中赋值一次，因此合法、不报错（CI 日志亦未对该字段报错，进一步印证根因定位准确）。
