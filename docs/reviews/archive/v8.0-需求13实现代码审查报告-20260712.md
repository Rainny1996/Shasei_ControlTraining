### ❌ 编译 error:C1 — ModeSelectionCard 语法错误

**位置**:`CoachView.swift:753`
```swift
.background(isSelected ? Color.accentColor.opacity(0.08) : Color(.systemGray6)))
```
**问题**:`.background(...)` 后有 **3 个** 右括号,但仅需 **2 个**(仅 `Color(.systemGray6)` 的 `)` 和 `.background(`的 `)` )。多余的 `)` 会导致 Swift 解析错误。

**修复**:
```swift
// 改为:
.background(isSelected ? Color.accentColor.opacity(0.08) : Color(.systemGray6))
```

### ⚠️ Feature gap:F1 — PlanItemDetailView 未传递 `initialMethodMode`

**位置**:`PlanItemDetailView.swift:52-58`

当前调用:
```swift
CoachView(
    initialMethod: method,
    planItemId: item.id,
    onPlanItemComplete: { planViewModel.markItemCompleted(item.id) }
)
```

**缺失**:未传入 `initialMethodMode:`。AC-13.6 要求计划项入口应预选该项保存的模式。

**应补充**:
```swift
CoachView(
    initialMethod: method,
    planItemId: item.id,
    initialMethodMode: method.trainingModes.first(where: { $0.id == item.modeId }),
    onPlanItemComplete: { planViewModel.markItemCompleted(item.id) }
)
```
