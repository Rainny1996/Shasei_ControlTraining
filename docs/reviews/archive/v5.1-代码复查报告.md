## 三、BUG-CT-11 退化详情（当前唯一剩余 P0）

**模型事实**：`CDCheckInRecord` 实体仅含 `id / date / checkInTime / trainingRecordId`（`ControlTrainingModel.xcdatamodel` 60-66 行），**无 `status`**。

**当前代码**（`DataViewModel.swift:208-212`）：
```swift
exports.checkInRecords = records.map {
    CheckInEntry(id: $0.id ?? UUID(), date: $0.date ?? Date(),
                 checkInTime: $0.checkInTime ?? Date(), status: $0.status ?? "valid")
}
```
- `$0.status`：`CDCheckInRecord` 无此属性 → 编译错误。
- `CheckInEntry` 结构体（241-245 行）无 `status` 形参 → 编译错误。

**修复方式**（删除 status 引用，因模型从未持久化该字段）：
```swift
CheckInEntry(id: $0.id ?? UUID(), date: $0.date ?? Date(),
             checkInTime: $0.checkInTime ?? Date())
```

修复后 `DataViewModel` 不再引用任何不存在的属性，主 Target 方可编译。
