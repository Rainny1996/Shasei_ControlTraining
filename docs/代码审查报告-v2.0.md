## 三、新发现回归缺陷

### 🔴 BUG-CT-07（回归）：S05 修复导致 `SecurityService` 编译错误

| 属性 | 内容 |
|------|------|
| 编号 | BUG-CT-07 |
| 等级 | **致命（编译阻断）** |
| 来源 | S05 建议修复（移除 SecurityService 内嵌 Keychain 方法）不完整 |
| 文件 | `SecurityService.swift` |

#### 问题描述

按 S05 建议移除了 `SecurityService` 中的 `saveToKeychain`/`loadFromKeychain`/`removeFromKeychain` 私有方法（原 L351-395），但**未将调用方迁移到 `KeychainService.shared`**，导致 7 处编译错误：

| 行号 | 调用 | 错误 |
|------|------|------|
| L102 | `saveToKeychain(key: passwordSaltKey, data: salt)` | `Cannot find 'saveToKeychain' in scope` |
| L103 | `saveToKeychain(key: passwordKey, data: hashedPassword)` | 同上 |
| L116 | `loadFromKeychain(key: passwordSaltKey)` | `Cannot find 'loadFromKeychain' in scope` |
| L117 | `loadFromKeychain(key: passwordKey)` | 同上 |
| L130 | `removeFromKeychain(key: passwordKey)` | `Cannot find 'removeFromKeychain' in scope` |
| L131 | `removeFromKeychain(key: passwordSaltKey)` | 同上 |

#### 修复方向

将 7 处调用替换为 `KeychainService.shared` 的对应方法：

| 原调用 | 替换 |
|--------|------|
| `saveToKeychain(key:, data:)` | `KeychainService.shared.save(data: data, forKey: key)` |
| `loadFromKeychain(key:)` | `KeychainService.shared.load(forKey: key)` |
| `removeFromKeychain(key:)` | `KeychainService.shared.delete(forKey: key)` |
