## App Store 描述

控训是一款专为男性设计的科学训练辅助应用，通过系统化的训练方法、智能计划制定和全面的数据分析，帮助您有效提升控制能力。

【核心功能】

• 训练方法库
- 涵盖凯格尔运动、停-动技术、挤压技术、呼吸训练、骨盆底肌训练等多种科学方法
- 每种方法配有详细原理说明、操作步骤和注意事项
- 按难度分级，适合不同阶段的用户

• 实时陪练
- 语音引导，如同私人教练在旁指导
- 多种训练模式：基础、渐进、间歇
- 呼吸引导动画，配合训练节奏
- 支持后台播放，锁屏不中断

• 智能计划
- 初始评估问卷，了解您的当前状况
- 基于评估结果智能生成个性化训练计划
- 根据训练数据动态调整计划难度
- 支持手动调整和训练提醒

• 每日打卡
- 一键打卡，记录训练完成情况
- 连续打卡激励，养成持续训练习惯
- 打卡日历，直观展示历史记录
- 补签功能，每月3次机会

• 训练复盘
- 训练结束后即时复盘问卷
- 历史记录查看和数据趋势分析
- 周/月复盘报告，追踪进步
- 文字备注，记录训练心得

• 状态分析
- 五维度能力评估：持久力、控制力、恢复力、呼吸配合、肌肉力量
- 能力雷达图，直观展示各维度水平
- 能力等级划分，清晰了解当前阶段
- 个性化改善建议和薄弱环节训练推荐

• 隐私保护
- Face ID / Touch ID 生物识别解锁
- 应用密码锁
- 后台切换自动模糊保护
- AES-256 数据加密存储
- 所有数据仅存储在本地设备

【适用人群】
- 希望系统提升控制能力的成年男性
- 追求科学训练方法的用户
- 注重隐私保护的用户
